WordCat help file

What's the goal of this application?

I had to merge a lot of Word documents located in different folders. These files could me modified almost anytime, so I had to develop this tool.
It takes the order you give into account for merging files, so you don't have to use some weird filenames to organize your final document.

Advantages

  • Files and directories with a name beginning with a dash are ignored.
  • Non-latin characters into Word documents are naturally supported.

How to use this application

  1. Select a target folder and an optional template (located in the same folder):
  2. Use the arrows to move up/down the selected file, or use the A-Z icon to sort them alphabetically:
  3. Once you've browsed each folder, use the Check button to verify whether all documents are referenced.
  4. If it succeeds, click on the Run button and select the method you prefer. The result will be stored in a file called merge.doc into the target directory's root.
    The application will automatically open it for reviewing. Always do some visual checks before printing it!

Known bug

Concatenating files with headers/footers may cause some problems.
Try removing them via the integrated tool (Tools → Remove headers/footers) after making a backup...
This bug is actually due to Microsoft Word's bad headers/footers management.
